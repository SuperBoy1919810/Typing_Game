# 打字练习游戏 - Typing Practice Game

一个功能丰富的Python打字练习游戏，支持中英文界面、多样化设置和音效反馈。

## 环境要求

- Python 3.6+
- Pygame库

## 安装

1. 确保已安装Python 3.6或更高版本
2. 安装Pygame：
   ```bash
   pip install pygame
   ```

## 运行游戏

```bash
cd e:\EDIT\20260511_Python课综合作业\TypingGame
python typing_game.py
```

## 游戏规则

1. 点击"开始游戏"按钮或按空格键进入游戏
2. 屏幕顶部会不断掉落字母（A-Z）和数字（0-9）方块
3. 玩家需要在字符掉落之前按下对应的按键
4. 正确按键消除字符并得分，同时播放提示音效
5. 失误则字符掉出底部
6. 游戏时间结束后显示统计和评级
7. **重要：游戏中请关闭中文输入法，否则无法正确识别按键**

## 功能特性

### 核心游戏
- 流畅的字符掉落动画
- 实时虚拟键盘高亮反馈
- 详细的游戏数据统计（得分、失误、失误率）
- 多级别评价系统（S/A/B/C/D级）

### 双语支持
- 支持简体中文和English界面
- 可在设置中随时切换语言

### 个性化设置
- **窗口分辨率**：多种分辨率可选（1024x768 至 1920x1080）
- **下落速度**：慢速/中速/快速/超快
- **游戏时长**：15秒至180秒多种选择
- **键盘提示**：开启/关闭虚拟键盘显示
- **音乐音量**：滑块调节（0-100），滑块旁显示当前数值

### 设置界面交互机制
- 下拉选项卡一次只能展开一个
- 展开新选项卡时，之前未操作的选项卡自动恢复原设置
- 音量滑块调整后，点击返回会恢复原始音量设置
- 音量滑块支持拖动调节，滑块旁实时显示数值

### 音效系统
- 按钮点击音效（800Hz清脆提示音，音量60%）
- 键盘按键消除音效（与按钮相同的提示音）
- 背景音乐播放（可选，需放置bgm.mp3文件）
- 所有音效音量可调

### 视觉效果
- 半透明背景框和边框
- 自定义背景图片支持
- 默认纯色背景（菜单-红色、游戏-蓝色、结束-绿色）
- 虚拟键盘带F、J、0三个按键的下划线标识

### ESC提前退出机制
- 游戏中途可按两次ESC快速结束
- 第一次按ESC显示提示，5秒内再按一次即可提前结算
- 提前结束不显示评级，只显示统计数据
- 虚拟键盘上的ESC键会以绿色高亮显示当前等待状态

## 操作说明

### 主菜单
- **开始游戏**：点击按钮或按空格键
- **设置**：点击进入设置界面
- **退出游戏**：点击并确认后退出

### 游戏界面
- **字母/数字键**：消除对应字符
- **ESC键**：第一次显示提示，第二次提前结束游戏

### 设置界面
- 下拉菜单：点击展开选项
- 音量滑块：拖动调节，数值显示在滑块右侧
- 应用/返回：点击确认或取消设置

## 自定义设置

### 背景图片
在项目根目录下创建 `images` 文件夹，放入以下图片：
- `menu_bg.png` - 开始菜单背景
- `game_bg.png` - 游戏进行时背景
- `gameover_bg.png` - 游戏结束背景

### 背景音乐
在项目根目录下创建 `music` 文件夹，放入 `bgm.mp3` 文件即可自动播放。

### 配置文件
`config.json` 会自动保存您的设置，包括：
- 分辨率、游戏速度、时长
- 语言选择
- 键盘提示开关
- 音乐音量

## 评级标准

| 评级 | 失误率 | 评价 |
|------|--------|------|
| S级 | < 3% | 完美无瑕 |
| A级 | 3% - 10% | 优秀 |
| B级 | 10% - 20% | 良好 |
| C级 | 20% - 40% | 合格 |
| D级 | > 40% | 需继续努力 |

**注意**：使用ESC提前退出时，不显示评级

## 常见问题

**Q: 运行游戏时出现 "No module named 'pygame'"？**
A: 运行 `pip install pygame`

**Q: 背景图片不显示？**
A: 确保图片放在 `images` 文件夹中，格式为PNG

**Q: 没有听到音乐？**
A: 确保 `music/bgm.mp3` 文件存在

**Q: 游戏中有中文字符无法识别？**
A: 确保关闭中文输入法再开始游戏
